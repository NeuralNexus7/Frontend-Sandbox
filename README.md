# 🎨 UI Animation Collection

A collection of creative and interactive UI animations built using **HTML**, **CSS**, and **JavaScript**.

This repository contains multiple frontend animation projects that can be used for learning, inspiration, or integration into modern websites.

## 🚀 Projects Included

### 📦 3D Cube Animation

* Interactive rotating 3D cube
* Pure HTML & CSS
* Smooth animations

### 🃏 3D Flip Card

* Front & back card flip effect
* Profile card design
* Hover interactions

### 🎠 3D Card Carousel

* Rotating card showcase
* Smooth transitions
* Responsive layout

### ✨ Hover Effects

* Modern button and card hover animations
* Clean UI interactions

### ⏳ Loading Animations

* Creative CSS loaders
* Lightweight and customizable

### 🎭 Additional UI Experiments

* Various animation concepts
* Creative frontend designs

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
